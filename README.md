# Technotran

Full-stack project: React (Vite) + Tailwind frontend, Node.js + Express backend.

## Structure

- `frontend/` — Vite + React 19, Tailwind CSS v4
- `backend/` — Express API (port `5000`)

## Quick start

**Frontend**

```bash
cd frontend
npm install
npm run dev
```

**Backend**

```bash
cd backend
npm install
npm start
```

Health check: `GET http://localhost:5000/api/health`
